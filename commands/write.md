---
description: "Write new content in a previously analyzed author's voice. Load a style profile and generate text matching all 24 dimensions."
---

Invoke the ghost-writer skill in WRITE mode.

1. Ask the user which style profile to load (or let them specify a file path).
2. Load the `{author}-style-profile.json` file.
3. Ask what they want written (blog post, email, essay, documentation, etc.).
4. Generate a draft following every dimension in the profile.
5. Self-audit the draft against all 24 dimensions.
6. Revise any dimensions that are off.
7. Do a final "uncanny valley" check.
8. Output the finished content.
