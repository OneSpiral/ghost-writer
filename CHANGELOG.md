# Changelog

## v1.1.0 (2026-03-17)

### Added
- Claude Code plugin format (`.claude-plugin/`) for marketplace installation
- Slash commands: `/analyze`, `/write`, `/compare`
- `install.sh` — auto-detect agent and install in one command
- `package.json` for plugin metadata
- `CONTRIBUTING.md` with contribution guidelines
- Issue templates (bug reports, feature requests)
- PR template
- GitHub Sponsors (`FUNDING.yml`)
- Social card for link previews
- Cross-link to [code-autopsy](https://github.com/OneSpiral/code-autopsy)

## v1.0.0 (2026-03-17)

### Initial Release
- 24-dimension forensic writing style analysis framework
- Analyze mode: feed writing samples → JSON profile + interactive HTML report
- Write mode: load profile → generate in author's voice with self-audit
- Example: Paul Graham style profile and interactive report
- Compatible with Claude Code, Codex CLI, Pi, OpenCode, and any agent
