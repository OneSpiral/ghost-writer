# Contributing to ghost-writer

Thanks for your interest! Here's how to contribute.

## Ways to Contribute

### 📝 Pre-built Author Profiles
Run ghost-writer on a well-known author and contribute the style profile + report as an example. Great first contribution.

### 🔬 New Dimensions
The current framework has 24 dimensions. If you think we're missing something important, open a feature request with examples.

### 📊 Report Design
The HTML report is the product. Visual improvements, better charts, accessibility fixes — all high-value.

### 🌍 Non-English Support
The 24 dimensions are language-universal in theory. If you have expertise in analyzing style in another language, help us validate and adapt.

## Pull Request Process

1. Fork the repo
2. Create a branch: `git checkout -b feat/my-change`
3. Make your changes
4. Test by running the skill on real writing samples
5. Submit a PR with a clear description

## Style Guide

- SKILL.md is the source of truth
- HTML reports must be self-contained (no external dependencies)
- Every dimension must be measurable, not just descriptive
- Before/after examples are required for new dimensions

## Code of Conduct

Be constructive and respectful. Writing is personal — critique the analysis, not the author.
